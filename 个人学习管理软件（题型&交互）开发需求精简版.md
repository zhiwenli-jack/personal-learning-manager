# 个人学习管理软件（题型&交互）开发需求精简版

# 一、核心目标

实现“用户输入资料-系统生成题目-答题测评-数据沉淀”全闭环，适配个人多方向学习测评需求，支持智能化评分与错题管理。

# 二、题型需求（核心必做）

- 客观题：预生成标准答案，支持系统自动评分

- 选择题：包含单选、多选，归为客观题体系

- 是非判断题：归为客观题体系，测评基础知识点

- 主观题：支持AI智能评分，含简答题细分类型

# 三、用户端功能（极简操作）

- 学习方向选择：支持编程、数学、语言、哲学、文学等大类选择

- 原始资料输入：支持上传/录入资料（作为题目生成依据）

- 题目评价：对系统生成题目进行“好/不好”二元评价

- 答题模式：支持限时、非限时两种模式，用户自主选择

- 评分查看：支持100分制、A/B/C/D等级制，可自主选择或默认

# 四、系统侧功能（自动化处理）

- 输入管理：存储、分类用户学习方向、原始资料，支持调取

- 资料处理：解析用户输入资料，提炼核心知识点（支撑题目生成）

- 题目生成：基于核心知识点，匹配题型体系自动生成题目

- 测验拟定：适配资料类型自动推荐答题模式，支持用户自主选择

- 评分处理：客观题自动匹配标准答案评分，主观题AI评分，输出对应结果

- 错题记录：自动抓取错题，按学习方向、知识点分类存储

- 正向反馈：结合答题结果、题目评价给出反馈，优化后续题目生成

# 五、核心约束

- 用户操作极简，核心功能3步内完成

- 评分逻辑清晰，AI评分需贴合主观题答题要点，客观题评分零误差

- 数据可追溯，错题、学习记录、资料可随时调取/查看
